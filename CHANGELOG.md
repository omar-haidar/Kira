# Changelog

- Updated Shizuku library to 13.1.5.
- Increased minimum SDK to 24, so the app no longer runs on Android 6.0 or lower.
- Switched root command execution to the libsu library.
- Added a Settings screen.
- Added a Full screen toggle in Settings.
- Added a theme selector in Settings and added white/black themes.
- Added a Dynamic colors toggle in Settings.
- Added a Permission mode selector in Settings.
- Added a Search bar in filter Dialog.
